# Multiobjective Dijkstra Algorithm

Run the Multiobjective Dijkstra Algorithm with graph files of cost dimension 3 (example [instances](https://github.com/maristanyPedro/targetedMosp/tree/v1.0.0/multidimensional/instances)).

## Usage

Dotnet:

```txt
dotnet run --project src/TargetedMosp.Cli/target.dotnet.dim.3/TargetedMosp.Cli.fsproj <path> <source> <target>
```

JavaScript:

```txt
dotnet fable src/TargetedMosp.Cli/target.javascript.dim.3/TargetedMosp.Cli.fsproj -o build
node build/Program.js <path> <source> <target>
```
