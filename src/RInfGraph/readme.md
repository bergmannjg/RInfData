# Graph Library

Find shortest path for operational points, assumes that graph is generated by RInfLoader.

## Fable

* add '"type": "module"' to file 'src/RInfGraph/build/.fable/fable-library.3.3.0/package.json'
* add fable directory to FSharpx.Collections nuget package till version 3.0.0 is released
* build: dotnet fable RInfGraph.fable.fsproj -o build
* usage: node build/Main.js path-to-graph-file "DE   HH;DE   BL"
